## 🚀 Project Overview

RASEED revolutionizes personal finance management by combining intelligent receipt scanning, automated expense tracking, and beautiful Material Design UI. Built with modern Flutter architecture and powered by Google's ecosystem, it offers a seamless experience for managing receipts, warranties, and financial insights.

## ✨ Key Features

### 🔍 **AI-Powered Receipt Scanning**
- **Smart OCR Technology**: Instantly extract data from receipt images
- **Multi-format Support**: Photos, PDFs, and document scanning
- **Automatic Categorization**: AI categorizes expenses intelligently
- **Real-time Processing**: Lightning-fast cloud-based parsing

### 💼 **Comprehensive Expense Management**
- **Digital Receipt Storage**: Secure cloud-based receipt vault
- **Expense Analytics**: Beautiful charts and spending insights
- **Category Tracking**: Automated expense categorization
- **Multi-currency Support**: Global currency handling with INR focus

### 🛡️ **Smart Warranty Management**
- **Warranty Tracking**: Never miss warranty expiration dates

### 🎯 **Family & Social Features**

## 🚀 Getting Started

### Prerequisites
```bash
# Install Flutter SDK
https://docs.flutter.dev/get-started/install

# Verify installation
flutter doctor
```

### Installation
```bash
# Clone the repository
git clone https://github.com/Harshavardhan-28/Raseed.git
cd Raseed

# Install dependencies
flutter pub get

# Configure Firebase (google-services.json included)
# Run the app
flutter run
```

### Firebase Configuration
1. **Google Services**: Pre-configured with `google-services.json`
2. **Authentication**: Google OAuth and Email/Password enabled
3. **Database Rules**: Secure Firestore rules implemented
4. **Storage Rules**: File upload permissions configured
